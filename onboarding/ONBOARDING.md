Welcome to the Spendly Contributor Cohort! Follow these steps carefully to set up your development environment.

---

## 1. Requirements
- Node.js LTS
- Git
- VSCode
- Android Studio (for emulator)
- GitHub account

---

## 2. Clone the Repository
```
git clone https://github.com/SpendlyHQ/spendly-app.git
cd spendly-app
```

---

## 3. Branch Workflow
- You cannot push to `main`
- You must create a branch:
```
git checkout -b feature-yourname-task
```
- Push normally; PR is required

---

## 4. Install Dependencies
```
npm install
```

---

## 5. Running the App
```
npm start
```

On Android:
```
a
```

---

## 6. Pull Requests
All PRs must:
- Target `dev` branch
- Follow the PR template
- Have clean commit messages
- Include screenshots if applicable

---

## 7. Communication
Join the Spendly Co-Developers WhatsApp/Discord group.
